How It Works
  • Add multiple recipients using the recipient input (supports Manager, Submitter, Site DLs)
  • Use Template Builder tab to create professional HTML emails for New Hire requests
  • Email notifications are queued in Azure Service Bus (not sent immediately)
  • Power Automate consumes messages from the queue and sends actual emails
  • Message metadata (NotificationType, Priority, Module, Trigger) is used for filtering and routing
  • Preview tab shows rendered HTML before sending
  • Check the Azure Service Bus dashboard or Power Automate logs to verify message delivery
